# Облік медикаментів (Курсовий проєкт 2024)

Програма для автоматизації обліку лікарських засобів, розроблена як курсовий проєкт у 2024 році. Додаток створено на платформі **.NET (WPF)** із використанням мови програмування **C#** та принципів об'єктно-орієнтованого програмування (ООП).

### 🚀 Основні функції:
* **Управління базою медикаментів:** додавання, перегляд та класифікація лікарських засобів за формами випуску (таблетки, сиропи, креми тощо).
* **Гнучкий пошук:** швидкий пошук медикаментів у базі за назвою, виробником або типом форми.
* **Авторизація:** наявність вікна доступу (паролю) для захисту даних.
* **Робота з виробниками:** облік інформації про фармацевтичні компанії (наприклад, ПАТ «Дарниця», ПАТ «Фармак»).

### 🛠 Технологічний стек:
* **Мова програмування:** C#
* **Інтерфейс (UI):** WPF (XAML)
* **Збереження даних:** Текстова база даних (структуровані файли `.txt`)
